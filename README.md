## これは何か

`TENCHO AI ECONOMY` の **L1 思想層 (Manifesto Layer)** ── 七箇条と三つの仁義を掲げる、静的な宣言文書。

- 形式: 単一HTMLファイル (`index.html`)
- 依存: Tailwind CSS (CDN) / Google Fonts / Vanilla JS
- ライセンス: CC0 1.0 (Public Domain) ── **改変・複製・再配布・商用利用すべて自由**

## なぜCC0なのか

第五箇条 **「情報の格差は奪い返す」** に従い、この宣言文書は誰のものでもない。フォークしろ。改変しろ。自分の経済圏のマニフェストに使え。それがこの経済圏の作法である。

## アーキテクチャ三層

```
L1  思想層 / Manifesto Layer  ← このリポジトリ
    Static HTML · Tailwind CDN · GitHub Pages
    https://tencho-ai-893.github.io/tencho-manifesto/  [LIVE · CC0]

L2  事業層 / Application Layer

L3  プロトコル層 / Protocol Layer  (予定)

```

L1 で叫び、L2 で運用し、L3 で仁義を貫く ── 単一スタックではなく責務分離による統合最適。

## 七箇条

1. AIは握るものではない。使い倒すものである。
2. 弱者から先に、AIを渡せ。
3. 義のないAIは、ただの暴力である。
4. 経済圏の天頂に、個人を立たせる。
5. 情報の格差は、奪い返す。
6. 仁義は、コードで貫く。
7. そして、叫び続けろ。AIを、使わせろ。

## 三つの仁義 (Doctrines)

- **義** ── 同志を、喰わない
- **仁** ── 利を、独り占めしない
- **勇** ── 声を、上げ続ける
